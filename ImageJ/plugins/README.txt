You can compile plugins within ImageJ using either the
Plugins>Compile and Run command or by pressing ctrl-r
(File->Compile and Run) in ImageJ's built in text editor.

The plugins contained in Input-Output>QuickTime_Plugins.jar
(Mac OS X only) are preinstalled as the File>Import>Using QuickTime,
File>Import>Video, File>Import>QuickTime Player and
File>Save As>QuickTime Movie commands.
